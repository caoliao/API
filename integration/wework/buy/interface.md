# 企微接口详解及购买

> 企微成员需要在企业微信环境中打开并使用草料二维码应用，就需要购买接口。多少个企微成员需要使用，就要购买多少个接口。
>
> 购买一个接口许可账号实际对应购买了一个接口激活码，管理员购买接口后，成员需要在企微平台激活接口后才可使用应用功能。

由于企业微信调整的收费规则，所有使用企业微信版草料应用的企业都需要根据使用成员数量来支付企微接口费用。详细见企业微信侧公告：[接口收费定价与细则](https://open.work.weixin.qq.com/wwopen/common/readDocument/38914)

## 1. 哪些成员及操作需要购买接口

**企微成员**在企业微信使用草料应用的任何操作都**需要**购买接口，详细如下：

1. 企业微信扫码登录草料电脑端后台

2. 在企业微信工作台中打开草料应用

3. 在企业微信中扫码查看二维码内容、填写表单、变更状态、查看动态档案、转发自己填写的表单数据给同事

4. 查看同事分享的表单数据或二维码、接收和查看消息提醒、对某条记录添加后续处理情况等等

企业微信成员外的用户在微信环境下完成的任何扫码、填表等操作，都**不需要**购买接口

## 2. 接口试用

- 企业微信平台提供接口试用期90天，自应用首次安装日起算（非版本付费日）。将草料应用删除重装，试用期不变。

- 到期后如需继续在企业微信中使用草料应用，则必须购买接口，购买方式如下。

## 3. 购买接口

> 如果同时需要购买版本费的，[请查看此流程](https://cli.im/help/83327)

### a. 根据需要的接口数量计算价格

| 接口数量区间 | 接口价格 |
|------------|---------|
| 1-5 | 5元/个/年 |
| 6-200 | 4元/个/年 |
| 201-500 | 3元/个/年 |
| 501-1000 | 2元/个/年 |
| 1001-10000 | 1元/个/年 |
| 10001及以上 | 面议 |

根据需求的接口数量及年数计算总费用，提供了[费用计算器](https://cli.im/qy-calc/index.html)，可输入数量及年数，自动计算接口总费用。

附：企微的[接口收费定价与细则公告](https://open.work.weixin.qq.com/wwopen/common/readDocument/38914)，对应的是基础账号。

**收费示例：**

企业A， 可见范围的企微成员人数为20人，使用2年，那至少需要购买20个接口费。

费用为：（5人×5元/个/年+15人×4元/个/年）×2=170元

> **注：接口购买后需激活，激活有效期 = 购买时长**，如购买1年，则需在1年有效期内，完成激活，激活后可继续使用1年使用时长。

### b. 支付费用

最终支付给企业微信，由草料代收取代开通，支持以下两种支付方式：

- 支付宝：zhifubao@nears.cn

- 对公转账：如果需要公对公转账，请[联系客服](https://cli.im/api/soboten/commonLink?channel_id=12)获取专属汇款账号

### c. 登记购买和开票信息

支付完成后**务必完成填写登记**：

- 登记中需填写企业ID，获取方式：登录草料后台（[https://cli.im/](https://cli.im/)），点击【设置-项目信息-账户信息】，找到企业ID并复制

- [点击进行登记填写](https://doc.weixin.qq.com/forms/AHYAQQdjAAsAdMAGwbuAAsidwqkeP4Qif)

- 我们会在1~3个工作日开通接口许可，发票会按登记信息开出。

![登记购买和开票信息](https://blogcdnimg.clewm.net/2024/02/image-1709189792823_17091897914322.png?x-oss-process=image/auto-orient,1/quality,q_50/format,jpg)

### d. 发票申请

平台客服将按照你登记的开票信息，在1~3个工作日内开出发票。开票信息分成两种

- **与版本费合并开票**：发票名称为【信息技术服务*二维码管理系统】，税率为13%。

- **单独的接口费开票**：发票名称为【信息技术服务*二维码管理系统接口费】，税率为6%。

### e. 后期增购

如企业中新增成员，需在原基础上增购接口数量的，费用会重新从第一级阶梯（1-5人）开始计算。

### e. 购买限制

最多仅支持可购买5年。

## 4. 购买后激活接口

**默认使用自动激活**，即：只要在企业微信中打开草料小程序，就激活。用户操作包括：打开转发消息、系统消息提醒、工作台点击草料应用、扫码登录等。

**但用企业微信扫码操作时不能自动激活**。所以如果已购接口，但仍提示"应用无法使用"未开通接口不能使用时，需实际使用人在工作台点击草料应用方可激活。

> 请注意，接口须在购买后的1年内完成激活。如没有在1年内进行激活，该接口将会过期，无法使用。

## 5. 接口有效期

**自接口激活日起开始计算**，这意味着，每个接口到期时间可能都不一样。

举例：企业一次购买了2个接口1年；

成员A于1月1日使用草料时自动激活了1个接口，该接口将于次年1月1日到期；
成员B于2月2日使用草料时自动激活了剩余的另1个接口，该接口将于次年2月2日到期。

## 常见问题

### 1、为什么之前不要接口费，现在要购买接口？

这是由于企业微信平台调整了收费规则。接口费用实际上是企业微信平台收取的，而草料仅承担代收费代开通的角色。需要在企业微信平台使用各类应用的企业都必须支付相应的接口费，这不仅限于草料应用，其他应用也如此。

只是其他部分应用会将接口费直接纳入版本费用中，这使得这些应用在企业微信版本费用相较于官网版本费用更高。而这部分提高的费用往往超出大部分用户企业实际需要的接口数和接口费。

草料则选择了一种更加公开透明的收费模式，将应用版本费和接口费分开计算。草料的应用版本费与官网完全一致，支付给草料，企微额外收取的接口费按照官方收费规则最终支付给企微平台，草料不赚取任何差价。

然而，这种收费模式的劣势在于购买流程相对繁琐，也希望得到理解。

以下是企业微信官方发布的接口费收费标准公告，供参考：[企微接口收费定价与细则](https://open.work.weixin.qq.com/wwopen/common/readDocument/38914)

### 2、增加成员后超出购买接口数，会影响原有成员使用应用吗？

**不影响。**

成员增加后，不会自动删减原先已激活使用接口中的成员。
原先已激活接口且在有效期内的成员可以正常使用应用，新增成员无法使用，需增购接口。

### 3、成员离职，可以自动更换接口使用人吗？

**不支持自动更换接口使用人。需手动调整继承。**

如接口已激活且在有效期内，原成员离职，需要继承接口给其他成员，可联系在线客服手动调整。

### 4、什么是可见范围，如何调整可见范围

企业微信中的应用可见范围是指哪些员工可以在企业微信工作台看到这个应用。员工数即可见范围人数。
可在企业微信后台中，找到草料二维码应用，修改可见范围，[查看操作流程](https://cli.im/help/83412)

### 5、如何查看应用安装时间？

可登录电脑端后台，【设置-项目信息】中查看「创建时间」

### 6、企微授权到期后有什么提示吗？

免费试用企微授权到期或者购买企微授权到期后，在到期前7天进行提示，到期后进行拦截。
